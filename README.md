# mechanism-driven-literature-scan
A simple, configurable prompt for generating a daily, mechanism-focused research paper summary.
## What is this?

This is a **reusable prompt** designed for daily literature scanning.
Instead of listing papers, it helps you:
- focus on mechanistic insights  
- filter out low-value noise  
- quickly identify papers worth reading  

## How to use
1. Copy the prompt from `prompts/daily_scan.md`
2. Paste it into your preferred agent such as Codex automation
3. Set it to run daily
4. Receive a structured report of selected papers

## Why this is useful

Most literature tools return too many papers.

This prompt tries to:
- reduce overload  
- highlight important mechanisms  
- provide quick research inspiration  

## Customize for your field

You can adapt this to any topic by changing:

- disease / research area  
- keywords  
- mechanism focus  

Example:
- Parkinson’s disease → neuroinflammation, microglia  
- cancer → T cells, tumor microenvironment  
- infection → immune response, cytokines  

## Example use case

Originally designed for:

**Parkinson’s disease neuroimmunology**

But it works for any research area with minimal modification.
